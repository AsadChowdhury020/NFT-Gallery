# 🎨 NFT Gallery App (React + Vite + Tailwind CSS)

## 📌 Project Description

This project is a modern NFT Gallery application built using **React.js**, **Vite**, and **Tailwind CSS**.  
The application showcases digital NFT artworks with important metadata such as:

- NFT Name
- Artist Name
- Price (BDT)
- Category
- Rarity
- Description
- NFT Image

Users can browse NFTs, filter them by artist, sort them by price, and view full NFT details on a separate details page.

This project demonstrates:

- React Component Architecture
- Props Passing
- State Management
- React Router Navigation
- Filtering and Sorting Functionality
- Modern UI Design using Tailwind CSS

## 🧠 Concepts Used

### Component-Based Architecture

- `App.jsx` → Route management
- `Home.jsx` → Homepage UI
- `NFTCard.jsx` → Reusable NFT card
- `Details.jsx` → NFT details page

### Props Passing

NFT data is passed from parent to child components using props.

### State Management

React `useState()` is used for:

- Filter state
- Sorting state

### Dynamic Routing

React Router is used for:

```jsx
/details/:id
```

This creates a unique details page for each NFT.
